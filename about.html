About ElectroHub

ElectroHub is a demo e-commerce frontend built with HTML, CSS, and JavaScript only. It showcases product browsing, cart, wishlist, and a mock checkout flow using localStorage.

You can edit products in assets/js/data.js and colors in assets/css/styles.css.
